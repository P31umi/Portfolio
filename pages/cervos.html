Process description

The flow diagram describes an overview of the flow/product line for the poultry process in the factory. This process recycles chicken/poultry into oils, proteins and water. There are essentially three types of elements to the diagram, process machines, storage tanks and pumps.

Between each machine there is generally a pump to move the material on to the next stage of the process.

The first node in the diagram is the intake and is not a machine as such. The rest are as follows:
● The crusher is the first machine and is set to on/off with a speed control.
● The material is then pumped into either preheater machine (or both, depending on the size of the load). These machines have settings for speed, steam pressure, and discharge speed. Other sensor readings are intake chute level and discharge temperature.
